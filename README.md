# ProyectoSwingAcademico

## Descripción del sistema

El sistema permite gestionar estudiantes, cursos, matrículas y calificaciones mediante una aplicación de escritorio desarrollada en Java Swing. Implementa operaciones CRUD, utilizando almacenamiento en memoria (listas), sin conexión a base de datos.

## Requisitos para ejecutar
Java JDK 8 o superior
NetBeans IDE (recomendado)
Sistema operativo Windows / Linux / Mac

## Instrucciones de uso
1. Abrir el proyecto en NetBeans
2. Ejecutar el formulario principal
3. utilizar las pestañas para navegar entre los diferentes módulos
4. utilizar los botones según la acción que quiera realizar
5. Cada módulo cuenta con su propia tabla para visualizar los datos correspondientes.
6. Cerrar la aplicación al finalizar el uso.

## Estructura de carpetas
src/
 |-- modelo/         Clases del sistema  
 |-- repositorio/    Manejo de datos en memoria
 |-- negocio/        Lógica y validaciones
 |-- presentacion/   Interfaces gráficas 
 |-- util/           Clases auxiliares
 └-- excepciones/    Manejo de errores personalizados

## Módulos desarrollados
Modelo: clases que representan los datos del sistema
Repositorio: almacenamiento en memoria usando listas
Negocio: reglas de validación y lógica del sistema
Presentación: interfaz gráfica en Java Swing
Util: funciones auxiliares del sistema
